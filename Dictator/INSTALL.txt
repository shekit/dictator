Dictator - Voice-to-Text Dictation App
======================================

INSTALLATION INSTRUCTIONS:

Option 1 - Quick Install (Recommended):
  1. Open Terminal
  2. Drag "install-dictator.sh" into Terminal
  3. Press Enter
  4. Double-click Dictator.app to launch

Option 2 - Manual Install:
  1. Right-click Dictator.app
  2. Click "Open" (not double-click!)
  3. Click "Open" in the security dialog
  4. App will launch

Option 3 - Terminal Command:
  Open Terminal and run:
  xattr -cr /path/to/Dictator.app

FIRST LAUNCH:
  - Follow the onboarding to grant Microphone and Accessibility permissions
  - These are required for dictation to work

  IMPORTANT - Model Download:
  - During onboarding, voice recognition models (~200MB) will download
  - This happens as part of the setup process with a progress bar
  - Takes 2-5 minutes depending on internet speed
  - Internet connection required for first setup
  - You cannot proceed until download completes
  - Once downloaded, models are cached and app works offline

REQUIREMENTS:
  - macOS 14.0 (Sonoma) or later
  - Apple Silicon (M1/M2/M3/M4)

FEATURES:
  - Hold fn key, speak, release to dictate
  - Optional LLM cleanup (Cloud or Local via Ollama)
  - Stats tracking, transcription history
  - Custom dictionary for technical terms

Need help? Report issues at: https://github.com/anthropics/dictator/issues
